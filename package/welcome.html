This installer contains packages that allow you to work efficiently with the srcML format:

  • Convert source code to srcML, whether it's a single file, entire project, code fragment, or individual statement
  • Convert srcML back to source code without losing whitespace or formatting
  • Perform queries on source code using XPath and RelaxNG
  • Perform transformations on source code using XSLT

By default, the srcML command-line client package is installed. You also have the option to install the libsrcml development and example packages, which will enable you to integrate srcML features into your application.

For further details, visit srcML.org
